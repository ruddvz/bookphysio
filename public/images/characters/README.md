# Character Asset Inventory

BookPhysio uses a small set of decorative physiotherapy character illustrations across the marketing experience.

## Style guidelines

- **Format:** transparent PNG
- **Minimum export:** 640×960px
- **Art direction:** soft 3D render, calm clinical styling, professional but approachable expressions
- **Palette:** teal-led scrubs and warm neutral backgrounds that fit the BookPhysio token system
