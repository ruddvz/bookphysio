## Steps

1. **Read today's daily note** at `daily/YYYY-MM-DD.md` using today's actual date.


2. **Check the inbox/** folder — list any unprocessed files found there.

3. **Load active project context** — read the most recently modified file in `projects/` or `C4K-Brain/ACTIVE.md` if working on Caught in 4K.

4. **Summarize top 3 priorities** for today based on recent notes, active projects, and any inbox items.

5. **Ask**: "What are we working on today?"
